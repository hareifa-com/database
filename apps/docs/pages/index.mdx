# منصة اللاعبين الشباب

منصة متكاملة مصممة لاكتشاف وتقييم وتطوير المواهب الكروية الشابة في مصر. توفر المنصة أدوات متقدمة للمدربين والكشافات والأكاديميات لتتبع وتطوير اللاعبين الشباب بناءً على معيار تقييم موحد.

## 🎯 الأهداف الرئيسية

- **اكتشاف المواهب**: توفير نظام متكامل لتحديد وتتبع المواهب الكروية الشابة
- **تقييم موحد**: تطبيق معيار تقييم موحد وعلمي لجميع اللاعبين
- **تطوير مستمر**: متابعة تطور اللاعبين وتقديم برامج تدريبية مخصصة
- **ربط الفاعلين**: تواصل فعال بين اللاعبين والمدربين والأكاديميات

## 🏗️ بنية المنصة

تتكون المنصة من عدة تطبيقات وحزم متكاملة:

### التطبيقات (Apps)
- **Web**: التطبيق الرئيسي بواجهة مستخدم حديثة
- **Docs**: توثيق المنصة وAPI

### الحزم (Packages)
- **Database**: مخطط قاعدة البيانات مع Drizzle ORM
- **Config**: الثوابت والإعدادات المشتركة
- **API Client**: عميل TypeScript للـ API
- **UI**: مكتبة المكونات المشتركة

## 🚀 البدء السريع

```bash
# تثبيت الاعتماديات
pnpm install

# تشغيل بيئة التطوير
pnpm dev

# بناء المشروع
pnpm build
```

## 📊 معيار التقييم

يعتمد معيار التقييم على أربعة محاور رئيسية:

1. **المهارات الفنية** (30%)
   - التحكم بالكرة
   - التمرير
   - التصويب
   - المراوغة
   - اللعب بالرأس

2. **الصفات البدنية** (25%)
   - السرعة
   - التحمل
   - القوة
   - الخفة

3. **الجوانب الذهنية** (25%)
   - التركيز
   - اتخاذ القرار
   - الثقة بالنفس
   - العمل الجماعي

4. **الفهم التكتيكي** (20%)
   - التموضع
   - قراءة اللعب

## 🌟 المميزات

### للاعبين
- إنشاء ملف شخصي شامل
- تتبع التقييمات والتقدم
- التواصل مع الأكاديميات

### للمدربين والكشافين
- نظام تقييم متقدم
- تقارير تحليلية
- أدوات متابعة المواهب

### للأكاديميات
- إدارة اللاعبين
- برامج تدريبية مخصصة
- تقارير أداء شاملة

## 📱 التقنيات المستخدمة

- **Frontend**: Next.js 14, React 18, TypeScript
- **Database**: PostgreSQL مع Drizzle ORM
- **Styling**: Tailwind CSS
- **Components**: شريط أدوات Radix UI
- **Documentation**: Nextra
- **Build Tool**: Turbo (Monorepo)
